# cleanup Hamlet Extension

This is a Hamlet Deploy extension.

See docs.hamlet.io for more information.

## Description
A docker host cleaner. 

Removes containers and images older than the specified delay and period.

## Provider
`shared`

## Aliases
- `_cleanup`

## Supported Types
- `service`
- `containerservice`
- `task`
- `containertask`
